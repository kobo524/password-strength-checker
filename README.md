# Password Strength Checker 

A Python project that checks how strong your password is and gives friendly feedback.

## Features

- ✅ Checks for:
  - Minimum 8 characters
  - Uppercase and lowercase letters
  - Numbers
  - Special characters (@$!%*?&)
- ✅ Assigns a strength level: Weak ❌ / Medium ⚠️ / Strong ✅
- ✅ Provides specific feedback on missing elements
- ✅ Optional SHA-256 hashing demonstration

## How to Run

1. Clone or download this repository
2. Run the script:

```bash
python password_checker.py
